template<typename T >
void VaAny::emplace ( T && value)
inline

Constructs a new value in the VaAny object, replacing any existing value.

Template Parameters
TThe type of the value to construct.
Parameters
valueThe value to construct in the VaAny object.

This method uses perfect forwarding to construct the value in-place. If the size and alignment of the type allow, the value is stored in the small buffer optimization (SBO) storage. Otherwise, it is allocated on the heap. Any previously stored value is destroyed before the new value is constructed.
