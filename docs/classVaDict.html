Detailed Description

template<typename K, typename V, typename Hash = VaDictHash<K>>
class VaDict< K, V, Hash >

dictionary (map) that maintains insertion order. This class implements a dictionary (key-value store) using a hash table with separate chaining.

Template Parameters
KThe key type (must support equality comparison and hashing).
VThe value type.
HashThe hash function type (defaults to DictHash<K>).
Note
it preserves the order in which elements were inserted.
