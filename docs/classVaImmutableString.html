for VaLib.

Note
VaImmutableString is immutable, meaning it is not a good choice if you perform many string operations.
If you need a mutable string, you can use VaString.

Accessing specific characters in the string.

Parameters
indexThe index of the character to return.
Warning
The char type in C++ is only 1 byte, so if the string contains non-ANSI characters, this may not work as expected.
Since VaImmutableString is immutable, this function returns a copy, not a reference or modifiable value.
Exceptions
Ifthe index is out of the string's bounds, the function throws an IndexOutOfRangeError
