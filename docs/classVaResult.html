Detailed Description

template<typename T, typename E = VaBaseError>
class VaResult< T, E >

A class representing a result that can either hold a value or an error.

Template Parameters
TThe type of the value.
EThe type of the error (default: VaBaseError).

This class uses a union to store either a value of type T or a pointer to an error of type E. The error is stored as a pointer to allow polymorphic behavior (i.e. inheritance). Make sure the error type E has a copy constructor if you plan to copy VaResult objects.
