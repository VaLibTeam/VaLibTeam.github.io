Detailed Description

template<typename E>
class VaResult< void, E >

Specialization of VaResult for the case when there is no value (void).

Template Parameters
EThe type of the error (default: VaBaseError).

This specialization is used when the successful result does not hold any value, only indicating success or failure. The error is still stored as a heap-allocated pointer to support polymorphism (i.e. inheritance).

Make sure the error type E has a copy constructor if you plan to copy VaResult objects.
