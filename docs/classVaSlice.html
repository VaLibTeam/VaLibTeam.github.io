Detailed Description

template<typename T>
class VaSlice< T >

view into a contiguous sequence of elements.

Template Parameters
TType of elements in the slice
Note
VaSlice doesn't own the data it points to - it's just a view.
Warning
The user must ensure the underlying data remains valid while VaSlice is used.
