Detailed Description

template<typename Head, typename... Tail>
class VaTuple< Head, Tail... >

Variadic recursive tuple implementation.

Template Parameters
HeadType of the current head element.
TailTypes of the remaining elements.
Note
This implementation recursively inherits from VaTuple<Tail...>.
